# be-beauty

Site E-commerce de cosmétiques de luxe (Laravel 12 + React).

## Fonctionnalités
- Gestion des articles (CRUD) avec images via Cloudinary.
- Espace Client & Admin sécurisés.
- Système d'emailing via Gmail.
- API REST consommée par une application React intégrée.

## Installation
1. `composer install`
2. `php artisan migrate`
3. `npm install` (à la racine)
4. `cd clientapi && npm install`
